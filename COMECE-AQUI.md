## 💡 Dicas Finais

✨ **Compile antes de tudo:** `npm run build`

✨ **Use PM2 para gerenciar:** `pm2 start json-server...`

✨ **Configure nginx corretamente:** Veja os exemplos nos guias

✨ **Teste a API:** `curl http://localhost:3001/usuarios`

✨ **Não esquece das permissões:** `sudo chown -R www-data:...`

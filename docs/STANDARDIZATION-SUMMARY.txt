TROUBLESHOOTING RÁPIDO:
================================================================================

  ❌ "ERROR 1064": Syntax error no script
     → Parar (Ctrl+C)
     → Restaurar backup
     → Verificar linha exata do erro no arquivo SQL

  ❌ "ERROR 1452": Foreign Key violation
     → Script já lida (SET FK_CHECKS = 0)
     → Se persistir: Rollback completo

  ❌ "connect ECONNREFUSED": Banco desconectado
     → Verificar: systemctl status mysql
     → Reiniciar: systemctl restart mysql
     → Confirmar: mysql -u root -p -e "SELECT 1;"

  ❌ Aplicação não conecta pós-padronização
     → Verificar: backend/src/config/entities.js atualizado
     → Verificar: .env tem credenciais corretas
     → Reiniciar backend: npm start

  ⚠️  Underscores ainda existem pós-script
     → Rodar: post-validate-standardization.php
     → Identificar tabelas afetadas
     → Executar ALTERs manuais para colunas restantes
